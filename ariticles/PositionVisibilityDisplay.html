Element Positioning, its Display and Visibility

By default HTML elements fall in to two categories for the way they render/lay in a document: i.e, Block & Inline

  • Block elements: Block elements take new line and occupy the available width. Eg: div tag, p tag, h1 - h6 tags. These elements by default have their display property set to 'block'.
  • Inline elements: Inline elements continue to be in line with other elements as they appear and occupy only required space. Eg: span tag, a tag, button tag. These elements by default have their display property set to 'inline'.

Display and Visibility

  • display:none - When the display property is set 'none' the element is not visible and it will no more be part of its parent container.
  • visibility:hidden - When the visibility property is set 'hidden' the element is not visible but it will still continue to occupy space in the parent container.
