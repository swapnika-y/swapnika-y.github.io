Function Currying

Java script does not throw any error when a function is invoked with out passing all or any of the arguments that a function is expecting. When argument when used in the function will remain undefined. Java script allows partial and no application of the expected arguments.

See the Pen GqXYBw by Rajkumar (@raajkumar-s) on CodePen.

Function Currying is constructing a function that allows partial application of arguments.That is when all of the arguments are sent expected result is obtained from function otherwise a function is returned when less number of arguments are sent.
