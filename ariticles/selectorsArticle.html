CSS Selectors

CSS selectors can be classified into following categories. Basic Selectors, Attribute Selectors, Combinator Selectors, Pseudo classes and Pseudo elements

Basic Selectors:

Basic selectors are
  • Id selectors: '#id-name' '#' char is used before name of the selector to indiacate it is a Id selector and then the nale of the Id is specified.
  • class selecors: '.class-Name' '.' char is used before name of the selector to indiacate it is a Id selector and then the nale of the Id is specified.
  • Type selector: give the type of the element such as div/p/h4 etc to select the elements and one can start with assigning CSS properties.
  • universal selector: the wild card used here to select all the elements is '*'.

Attribute Selectors:

Attribute selectors are used to select the elements based on the attribute property and also based on its values( monkeynewkey key keys keying monkey) under attribute selectors following are options.

For example if DOM has few div elements which we want select having attribute 'class' and having multiple values(like newkey key keys keying and monkey) with space separated.

  • [class*="key"] is used to select all the div elements with attribute 'class' having 'key' among or as a part on any of the values(newkey key keys keying)
  • [class^="key"] is used to select all the div elements with attribute 'class' having 'key' in the begining of one of the values(key keys keying)
  • [class$="key"] is used to select all the div elements with attribute 'class' having 'key' in the ending of one of the values(key monkey)
  • [class~="key"] is used to select all the div elements with attribute 'class' having 'key' as exactly one of the values(key)

Combinator Selectors

Combinators have following operators to select the elements which are child/decendents or adjacent to a particular element.

  • Decendent Selector (space)

  • Child Selector(>)

  • Adjacent sibling Selector(+)

  • General sibling Selector(~)

CSS Pseudo Elements

CSS Pseudo Elements can be used to style the specific parts of an element such as:
  • ::first-letter - to style first letter of text
  • ::first-line - to style first line of text
  • ::before - to style and to insert some content before the actual content of an element
  • ::after - to style and to insert some content after the actual content of an element and
  • ::selection - to style the user selected text
These cannot be expressed as html. Double colon is used as a notation for pseudo elements from CSS3(To distinguish with Pseudo Classes). Single colon was used for both pseudo-classes and pseudo-elements in CSS2 and CSS1. For backward compatibility reasons Single colon is supported for Pseudo Elements in CSS3.
