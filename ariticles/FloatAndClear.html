Floating and arranging elements block elements

Block elements like div, p, header, section, aside etc always take a new line when created and occupy the entire width available if width is not specified. When it is required to arrange the block elements linearly thai is horizontally one next to other the block float property is made to set to 'left' or 'right'. when the elements are floated they are moved out of the container layout, hence if a border or background is applied to container it will not be seen surrounding the floated elements.

To fix this an additioan elemt is added along which has a CSS clear property set to either left right or both.
