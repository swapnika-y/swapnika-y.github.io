Position property

Position property: 'position' property is used to specify the positioning of an element. There are four basic values to define the positioning method to be adopted by the element they are: 'static', 'relative', 'absolute' and 'fixed'.

  • position:static - This is the default positioning value set to any element. When set to 'static' the top left bottom and right will have no effect on the positioning of the element. The element will occupy the space of its width x height in its container. The element will be positioned at its default position depending on its type(Block or Inline)

    See the Pen oLoQPr by Rajkumar (@raajkumar-s) on CodePen.

  • position:relative - When set to 'relative' the top left bottom and right properties of element will take effect with respect to its normal position of element in parent container, but will not effect the parent container element to resize or shuffle itself. It will continue to occupy its space in parent container.

    See the Pen Position:relative by Rajkumar (@raajkumar-s) on CodePen.

  • position:absolute - When position of an element is set to 'absolute', element will not occupy any space it its container and it will be moved out of the container's layout. The element gets attached to one of its positioned ancestor container(having the position set as 'relative'/'absolute'/'fixed').
