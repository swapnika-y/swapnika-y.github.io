Element Positioning, its Display and Visibility

By default HTML elements fall in to two categories for the way they render/lay in a document: i.e, Block & Inline

  • Block elements: Block elements take new line and occupy the available width. Eg: div tag, p tag, h1 - h6 tags. These elements by default have their display property set to 'block'.
  • Inline elements: Inline elements continue to be in line with other elements as they appear and occupy only required space. Eg: span tag, a tag, button tag. These elements by default have their display property set to 'inline'.

Display and Visibility

  • display:none - When the display property is set 'none' the element is not visible and it will no more take any space in its parent container.
  • visibility:hidden - When the visibility property is set 'hidden' the element is not visible but it will still continue to occupy space in the parent container.

Position property

Position property: 'position' property is used to specify the positioning of an element. There are four basic values to define the positioning method to be adopted by the element they are: 'static', 'relative', 'absolute' and 'fixed'.

  • position:static - This is the default positioning value set to any element. When set to 'static' the top left bottom and left will have no effect on the positioning of the element. The element will occupy the space of its width x height in its container.
  • position:relative - When set to 'relative' the top left bottom and left properties of element will take effect with respect to its initial position in parent container, but will not effect the parent container element to resize or shuffle itself. It will continue to occupy its space in parent container.
  • position:absolute - When position of an element is set to 'absolute', elementwill not occupy any space it its container. The element gets attached to one of its parent container having the position set as 'relative'/'absolute'/'fixed'. if none of its parent have position set to it the it is attached to the body element.
  • position:fixed - When position of an element is set to 'fixed', the element gets attached browser's viewport. Will not occupy any space its container.
